------------------------
HOW TO USE THE DATABASE:
------------------------

Import the database "webhotel.sql" from the project folder into phpMyAdmin

To enable database access you need to create a new
user account with the following details:
    - user name: web
    - host name: ever host (%)
    - password: web
    - add the database "webhotel" to the user and give all privileges (rights)
